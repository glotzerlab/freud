========
Tutorial
========

This tutorial provides a complete introduction to **freud**, including information on all the core classes in the package and how they can be used in concert.
The tutorial starts with a basic overview of core concepts to freud, followed by a demonstration of a few core features.
Detailed per-module introductions are left for the `examples`_.
